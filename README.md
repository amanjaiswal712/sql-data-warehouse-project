## 🚀 Project Overview

This project focuses on building a **complete data pipeline** — from raw data to business insights.

It covers:

- Designing a **data warehouse**
- Creating **ETL pipelines**
- Performing **data cleaning & transformation**
- Building **analytical models**
- Generating **business insights using SQL**

👉 The goal was to understand how raw business data becomes useful for decision-making.

---

## 🏗️ Data Architecture

I used the **Medallion Architecture** approach:

- **Bronze Layer** → Raw data stored as-is (CSV → SQL Server)  
- **Silver Layer** → Cleaned and transformed data  
- **Gold Layer** → Final analytical model (Star Schema)

This structure helped me understand how data flows in real industry systems.
